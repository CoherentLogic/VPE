Commands Cheatsheet **

What is it? VPE consists of a number of integrated programmer utilities that will increase productivity and decrease the number of key strokes required to complete normal programming tasks in the VISTA environment.

What should I download? If you are downloading to a PC, select 'Download VPE' in the Contents box to retrieve the entire VPE package as a self-extracting zip (.exe file). (Depending on how your browser is configured, you may need to use a right mouse click on the Download VPE link to access a menu option to save the download as a file on your computer.)

Support of GT.M began with version 12.
